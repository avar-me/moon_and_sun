# Луна и Солнце

**Практическая астрономия мусульманского календаря**

📖 Читать онлайн: [avar-me.github.io/moon_and_sun](https://avar-me.github.io/moon_and_sun/)

## О книге

Книга о практическом применении задач прикладной астрономии для составления мусульманских календарей и вычисления времени намазов.

### Часть I. Движение небесных тел
Солнечная система, орбиты Земли и Луны, системы небесных координат.

### Часть II. Вращение Земли и время намазов
Вычисление времени пяти ежедневных намазов, уравнение времени, различия между мазхабами (Ханафи vs другие для Аср), конвенции для Фаджр и Иша (MWL, ISNA, Египет, Умм аль-Кура и другие), особые случаи высоких широт.

### Часть III. Лунный календарь
Фазы Луны, вычисление новолуния, критерии видимости молодого полумесяца (хилял), определение начала и конца лунного месяца, календарь Хиджра.

## Сборка локально

```bash
# Установить mdBook
brew install mdbook    # macOS
# или cargo install mdbook

# Собрать
mdbook build

# Запустить локальный сервер с live-reload
mdbook serve
```
